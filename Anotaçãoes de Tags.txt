<h1>, </h1> = Serve para colocar o Titúlo principal-HTML

<p>, </p> = Serve para colocar um Parágrafo-HTML

<img> </img> = Serve para colocar uma imagem-HTML

<link:css>= Serve para colocar uma pagina CSS-HTML

<br> </br> = Serve para fazer uma quebra de linha-HTML

link:favicon = Serve para adicionar um favicon-HTML

<a>, </a> = Serve para adicionar links-HTML

<picture>, </picture>/ <source:midia> = Serve para adicionar uma segunda imagem, se a primeira falhar-HTML

<header>, </header> = Serve para definir um Cabeçario-HTML

<nav>, </nav> = Serve para definir a Navegação-HTML

<artcle>, </article> = Serve para definir o Artigo pricipal-HTML

<aside>, </aside> = Serve para definir o Artigo adicional ou secundario-HTML

<main>, </main> = Serve para definir a parte Pricipal-HTML

<footer>, </footer> = Serve para adicionar um roda-pé-HTML

<ul>, </ul> = Serve para adicionar uma Lista-HTML

<li>, </li> = Serve para definir a Organização de uma lista-HTML

<abbr>, </abbr> = Serve para fazer uma abreviação-HTML

<strong>, </strong> = Serve para Destaca-HTML

<title>, </title> = Serve para adicionar um titulo para seu site-HTML
